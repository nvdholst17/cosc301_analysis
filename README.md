# COSC 301 - Analysis Project

To recreate all the analysis, you need to do the following:

- Download the dataset from Kaggle at https://www.kaggle.com/datasets/rohanrao/formula-1-world-championship-1950-2020

- Run the provided [MySQL Formula 1 Schema](FinalAnalysis/Files/formula1Schema.sql) to create the database

## How to recreate Q1

1. Open Tableau and load in each table(from the MySQL database) as a data source

2. Run [Tableau Pit Stop Graphs File](PitStopAnalysis/301projAnalysis.twb) to recreate all graphs

## How to recreate Q2

1. Download the [Google Colab NoteBook](CircuitAnalysis/Reproduce%20the%20analysis/Q2_colab.ipynb)

3. Save the dataset files (CSV from Kaggle) and the NoteBook in Google Drive

4. Save a copy of the Colab NoteBook

4. Update the path to csv files defined inside the copied NoteBook 

5. Run the NoteBook

## How to recreate Q3

1. Open Tableau and load in each table(from the MySQL database) as a data source

2. Run [Tableau Regulation Graph File](RegulationAnalysis/RegulationGraphs.twb) to recreate all graphs
